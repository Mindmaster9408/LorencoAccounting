# The Neuro-Coach Method - Coaching App

A comprehensive 16-step coaching journey application for personal development and transformation.

## Features

- **16-Step Journey** across 3 phases
- **Interactive Exercises** for each step
- **AI Coach Integration** for guided support
- **Progress Tracking** with visual dashboard
- **Data Persistence** with localStorage and backup system
- **Responsive Design** for all devices

## Journey Phases

### Phase 1: Foundation (Steps 1-6)
1. 4 Quadrant Exercise
2. Present-Gap-Future
3. Flight Plan
4. Deep Dive
5. Ecochart
6. Assessments

### Phase 2: Processing (Steps 7-11)
7. The Dashboard
8. Psychoeducation
9. MLNP (Multi-Level Neuro Processing)
10. Reassess
11. Revisit

### Phase 3: Integration (Steps 12-16)
12. The Dream-Spot
13. Values & Beliefs
14. Success Traits
15. Curiosity, Passion, Purpose
16. Creativity and Flow

## Getting Started

1. Open `index.html` in a web browser
2. Create a new client profile
3. Begin the 16-step journey

## Technology Stack

- Pure HTML5, CSS3, JavaScript (ES6 Modules)
- No external dependencies
- LocalStorage for data persistence
- PostgreSQL backend ready for production deployment

## Deployment

### Current: GitHub Pages (Static)
- Accessible from any device with internet
- Data stored in browser localStorage
- Use backup/restore feature for data safety

### Future: Production Deployment
- PostgreSQL database for permanent storage
- Multi-user authentication
- Deploy to Railway, Render, or similar hosting

## Data Backup

The app includes automatic backup features:
- Auto-backup on every save
- Manual backup download (JSON format)
- Restore from backup file
- Access via "Backup & Restore" page
